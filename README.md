# Вычислитель отличий

Консольное приложение для поиска отличий в конфигурационных файлах.

####  Возможности

+ Поддержка файлов формата ***json***, ***yaml***, ***ini***
+ Генерация отчета в виде ***plain*** и ***pretty***

#### Установка

```
python3 -m venv venv

source venv/bin/activate

python3 install setup.py
```
